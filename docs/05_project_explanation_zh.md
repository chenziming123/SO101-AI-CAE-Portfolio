# SO-101 AI 辅助机械臂结构项目说明

## 项目概述

本项目基于开源 SO-101 follower arm，构建 AI 辅助机械结构改进流程。流程先复现官方 CAD/URDF，在 PyBullet 中验证 baseline，然后做工作空间、静载力矩和结构瓶颈分析，确定 `upper_arm_link` 是第一轮优化对象。之后使用 AI 辅助生成参数化 CAD V1，完成 STEP/STL 导出、URDF smoke test 和力矩对比；再用 STEP B-Rep 圆柱特征做孔位校核，发现 V1 装配孔系不足，于是迭代 V2，补充舵机安装孔、沉孔和肩部定位孔，并完成二级孔位校核和仿真验证。

## 重点数据

- 官方 upper arm 体积：117328.19 mm^3，PLA 粗估质量 145.49 g。
- V1 体积：93885.71 mm^3，相对官方约 -19.98%。
- V2 体积：98626.10 mm^3，相对官方约 -15.94%。
- V2 二级装配孔位校核：PASS。
- V2 接回 URDF 后，2001 个姿态最大末端位置差：0.000000e+00 m。
- 100g payload 下，官方 `shoulder_lift` p95：1.1440 Nm，V1：1.1268 Nm，V2：1.1303 Nm。

## 工程能力

- 开源机器人项目中的 CAD、STL、URDF 和仿真资源解析。
- 机械臂结构模块拆解、受力分析和结构瓶颈定位。
- AI 辅助参数化建模与 STEP 特征级孔位校核。
- CAD 结构件回接 URDF/PyBullet，并检查运动链是否被破坏。
- 设计取舍分析：V1 更轻，V2 更接近真实装配，V3 修正孔系余量。
- 建模问题记录与修复：V2 初版 pad 覆盖主孔，通过调整建模顺序解决。

## 当前边界

- 当前还不是生产级零件，后续需要标准件装配、干涉检查、FEA、打印方向和真实装机验证。
- 当前惯量是 screening 方法，按体积比例缩放 URDF 质量和惯量，不是最终材料标定。
- 当前 visual mesh 已接回 URDF，collision mesh 和完整物理接触仍需进一步完善。
