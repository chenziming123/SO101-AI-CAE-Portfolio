# SO-101 upper arm 面试讲述稿（Official / V1 / V2 / V3）

## 30 秒版本

我基于开源 SO-101 follower arm 做了一个 AI 辅助机械结构改进项目。先复现官方 baseline，再做工作空间、静载力矩和结构瓶颈分析，确定 upper arm 是第一轮优化对象。随后我用 AI 辅助生成参数化 CAD，做了 V1 轻量化、V2 装配接口增强、V3 孔系余量修正三轮迭代，并把每一版都接回 STEP 校核和 URDF/PyBullet 验证。

## 1 分钟版本

项目不是只生成一个外观模型，而是围绕真实开源机械臂做工程闭环：官方 baseline 能加载、能运动、能分析；V1 先把 upper arm 做轻；V2 补上舵机孔、沉孔和肩部夹紧孔，让它更像真实可装配结构；V3 则针对装配检查中暴露的孔系余量问题再次修正。最后我把 V3 接回 URDF/PyBullet，确认关节链和末端运动学没有被破坏，并用同口径静载力矩对比证明 `shoulder_lift` 负担仍低于官方 baseline。

## 你可以强调的能力

- 能把开源机器人项目拆成 CAD、URDF、工作空间、力矩和装配接口几个层次。
- 能用 AI 辅助参数化建模，但不会停留在“生成出来就算完成”。
- 能做 STEP 特征级孔位校核，找到真正的装配风险。
- 能把 CAD 结果接回仿真，做同口径的运动学和力矩回归。
- 能解释设计取舍：V1 更轻，V2 更完整，V3 更可靠。

## 三个关键数字

- 官方 upper arm 体积：117328.19 mm^3。
- V1 / V2 / V3 体积：93885.71 / 98626.10 / 101891.50 mm^3。
- 100g payload 下 `shoulder_lift` p95：官方 1.1422 Nm，V1 1.1254 Nm，V2 1.1288 Nm，V3 1.1313 Nm。

## 面试时可以主动说的边界

- 目前还没有做真实 FEA、打印方向、螺钉预紧和实物装机。
- 目前惯量是 screening 值，不是最终材料标定。
- 这条 upper arm 线已经完成闭环，下一步可以把同样方法复制到 lower arm。
