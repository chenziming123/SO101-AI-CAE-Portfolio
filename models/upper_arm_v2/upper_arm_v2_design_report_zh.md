# upper_arm_link V2 AI 辅助结构迭代设计报告

## 这一步在做什么

Step 11 在 `upper_arm_v1` 的基础上生成 `upper_arm_v2`。V1 已经证明主关节孔、中心距和 URDF 仿真链路可行，但它的装配特征还比较少。V2 的目标是把模型从“概念轻量化结构”推进到“包含明确装配孔系的结构件”。

## 为什么做这一步

V1 的主要价值是建立了参数化 CAD、STEP/STL 导出、URDF smoke test 和初步力矩收益验证。但 V1 的问题也很明确：

- 只有主轴孔、凸台和轻量化孔。
- 没有明确的舵机安装孔系。
- 没有肩部侧向夹紧/定位孔。
- 不能充分体现机械结构设计中最重要的装配接口思维。

因此 V2 的重点不是继续追求更轻，而是补充可解释、可校核的孔位和安装结构。

## V2 相比 V1 改了什么

1. 保留 V1 的 shoulder-to-elbow 主关节中心距：116.000 mm。
2. 保留 shoulder 与 elbow 主孔半径：4.200 mm。
3. 保留 shoulder 与 elbow 外凸台结构。
4. 在肘部 pad 上增加 2x2 舵机/连接件安装孔：
   - 通孔半径：1.600 mm。
   - 孔距：9.900 mm x 9.900 mm。
   - 中心位置：x=105.000 mm。
   - 增加 2.700 mm 半径的沉孔/让位特征。
5. 在肩部 pad 上增加两个侧向夹紧/定位孔：
   - 通孔半径：2.000 mm。
   - 位置：x=-8.000 mm，z=±10.500 mm。
6. 调整 web 与 rib：
   - web 厚度从 2.8 mm 增加到 3.0 mm。
   - web 高度从 26.0 mm 增加到 28.0 mm。
   - rib 厚度从 5.0 mm 增加到 5.5 mm。
7. 轻量化孔半径从 6.0 mm 调整为 5.4 mm，为新增装配孔和局部加强留出材料。

## 设计取舍

V2 相比 V1 体积增加约 4740.39 mm^3，PLA 粗估质量从 116.42 g 增加到 122.30 g，增幅约 5.05%。这不是设计退步，而是为了补充装配孔位、沉孔和局部加强 pad。

对机械设计项目来说，V2 的逻辑是：

- V1：先证明能轻量化和接回仿真。
- V2：补足装配接口，让结构更接近真实工程零件。

也就是说，V2 是用少量质量换取装配表达能力和接口可靠性。

## 中间遇到的问题和解决方式

第一次生成 V2 后，新增的舵机孔和肩部定位孔能被 STEP 检测到，但肩部主孔没有通过校核。

原因：

- 肩部加强 pad 与 boss 融合后，把原本的主轴孔特征覆盖/填充了。
- 这说明加强结构可能破坏关键装配接口。

解决方式：

- 在所有 pad、rib、boss 融合完成后，最后统一重新切削 shoulder 和 elbow 主轴孔。
- 这样可以保证任何加强结构都不能覆盖主孔。

这个问题本身很适合写进作品集，因为它体现了真实 CAD 迭代中的工程检查：不是生成完模型就结束，而是通过孔位校核发现问题，再修复建模顺序。

## 验证结果

几何验证：

- 官方 upper arm STL 体积：117328.19 mm^3，PLA 粗估质量 145.49 g。
- V1 体积：93885.71 mm^3，PLA 粗估质量 116.42 g。
- V2 体积：98626.10 mm^3，PLA 粗估质量 122.30 g。
- V2 相对官方仍减少约 15.94% 体积。
- V2 相对 V1 增加约 5.05% 体积。
- V2 STL 为 watertight。

孔位校核：

- V2 主关节接口复核：PASS。
- V2 新增装配孔系校核：PASS。
- V2 二级装配孔位校核总体：PASS。
- V2 STEP 去重后圆柱特征数从 V1 的 7 个增加到 21 个。

## 当前边界

V2 仍然不是生产级最终装配认证，原因是：

- 官方 STEP 和 V2 的完整装配基准还没有做严格 mating transform。
- 还没有引入舵机、螺丝、轴承等标准件模型做真实干涉检查。
- 还没有进行真实 3D 打印、公差补偿和装机测试。

因此当前准确表述是：V2 已完成 CAD 特征级二级孔位校核，但仍需要标准件装配和干涉检查。
